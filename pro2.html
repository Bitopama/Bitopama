Movies dashboard

Goal

The primary goal of this project was to analyze a comprehensive movie dataset to uncover trends in genres, runtime, approval ratings, and profitability. The objective was to create an interactive dashboard that provides insights into movie performance, helping stakeholders make data-driven decisions in the entertainment industry.

  • View on GitHub

Background Information

The dataset contained detailed information on various movies, including title, genre, runtime, release date, budget, revenue, and ratings. Data preprocessing was essential to resolve inconsistencies and tag movies with the same title by their production date, ensuring data accuracy. The cleaned dataset served as the basis for analyzing key metrics like return on investment and identifying popular genres and top-rated directors.

Dashboard Design and Creation

After preprocessing, I used DAX to create new measures that calculated return on investment and rated directors based on average movie ratings. The dashboard provides dynamic insights into movie performance by enabling filters such as genre, year, and rating. Key visuals include trends in average runtime over the years, popularity of genres, and a comparison of approval ratings against profitability.

The dashboard also features year wise comparisons of key metrics, highlighting trends in movie production and profitability. Monthly box office trends offer insight into seasonal patterns, with the data segmented by genre to identify preferences over time. Filters allow users to explore metrics based on criteria like genre and budget, helping stakeholders understand what factors influence success in the movie industry.

Further analysis examines the relationship between approval ratings and profitability, identifying genres and directors with the highest returns. The dashboard also highlights the most popular genres and the top-performing directors based on ratings, providing valuable insights into audience preferences and industry trends.

Conclusion

The final dashboard provided an interactive overview of the movie industry, allowing users to explore trends in genres, director ratings, and financial performance. By presenting these insights in an accessible format, the dashboard makes complex data available to stakeholders, enabling more informed decision-making.
